SigSync is a project that was written in python, the purpose of the project was to automate the way we created signatures in outlook in order to this the program first must pull from "active directory" essentially a database of attributes for users and laptops once it has pulled the main attrbiutes; name, title, office address, office name it will use those to create a signature in outlook.

Proxmox VE

Proxmox is a virtual enviroment OS that enables the ablity to launch multiple virtual machines in one single server, the OS sets up its own web server to allow for easy access and administration. PRoxmox is a free opensource solution and is very stable with the additions of first party backups, High Availabity and more!
